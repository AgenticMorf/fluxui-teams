# Installation

## Requirements

- PHP ^8.2
- Laravel ^11.0 or ^12.0
- Livewire ^3.0 or ^4.0
- livewire/flux ^2.0
- jurager/teams ^3.0

## Composer

```bash
composer require agenticmorf/fluxui-teams
```
